Scalable CIS

Integration of Service management with Assigned Tasks Using Customer Interaction System

Abstract

Managing and resolving customer issues is one of the main components of the Maintenance Phase of SDLC. This process is simplified with the help of Customer Interaction Systems. Organisations making use of such tools must adapt the systems based on the number of clients, as Current systems are far from flexible or scalable. The system proposed solves the problem of scalability and flexibility by integrating a document oriented NoSQL database which allows for data to be semi-structured which improves the overall throughput of the application.
